PowerBI (Feel free to skip this part. I did it as an example! Focus on the rest of the sections!)

Power BI is a business analytics solution that lets you?visualize?your data and share insights across your organization, or embed them in your app or website. Connect to hundreds of data sources and bring your data to life with live dashboards and reports.

Data wrangling, sometimes referred to as data munging, is the process of transforming and mapping data from one "raw" data form into another format with the intent of making it more appropriate and valuable for a variety of downstream purposes such as analytics.

The list of data sources for Power BI is extensive, grouped into the following:

  1. Files: Data can be imported from Excel (.xlsx, xlxm), Power BI Desktop files (.pbix) and Comma Separated Value (.csv).
  2. Content Packs: It is a collection of related documents or files that are stored as a group. In Power BI, there are two types of content packs, firstly those from services providers like Google Analytics, Marketo or Salesforce and secondly those created and shared by other users in your organization.
  3. Connectors to databases and other datasets such as Azure SQL, Databaseand SQL, Server Analysis Services tabular data, etc.

To do basic calculation and data analysis on data in power pivot, we use Data Analysis Expression (DAX). It is formula language used to compute calculated column and calculated field. DAX works on column values. DAX can not modify or insert data. We can create calculated column and measures with DAX but we can not calculate rows using DAX.

Power query is a ETL Tool used to shape, clean and transform data using intuitive interfaces without having to use coding. It helps the user to: Import Data from wide range of sources from files, databases, big data, social media data, etc; join and append data from multiple data sources. Shape data as per requirement by removing and adding data.

A new programming language is used in power query called M-Code. It is easy to use and similar to other languages. M-code is case sensitive language.

Front End Web App Development

HTML (Hypertext Markup Language) is the fundamental language used to create web pages. HTML5, the latest standard, supports multimedia elements and interactive features. It structures all the essential parts of a webpage, such as navigation, content, and other elements through tags like <header>, <footer>, <article>, and <section>, enhancing accessibility and SEO.

CSS (Cascading Style Sheets) styles the visual presentation of web pages. It controls layouts, colors, and fonts, and is crucial for responsive design. CSS3 includes features like Flexbox for complex layouts, CSS Grid for two-dimensional layout design, and custom properties (CSS Variables) for reusability and maintainability. It also includes pseudo-classes and pseudo-elements for fine-grained styling control.

JavaScript is a dynamic scripting language essential for adding interactive features to websites. It allows developers to manipulate the DOM, handle events, and execute asynchronous calls for seamless user experiences. Modern JavaScript (ES6 and beyond) includes features like arrow functions, promises for handling asynchronous operations, and modules for better code organization. Frameworks like React, Angular, and Vue simplify complex application development.

Excel Concepts

Conditional Formatting in Excel allows you to automatically apply formatting—such as colors, icons, and data bars—to cells that meet specific criteria. This feature is incredibly useful for highlighting trends, exceptions, and setting visual alerts based on the data's value.

Data Filtering helps manage large sets of data by displaying only the information that meets specific criteria. This tool is essential for focusing on relevant data without altering or removing non-matching data from the dataset.

Sorting and Filtering are fundamental data management tools in Excel. Sorting rearranges the data in your spreadsheet based on one or more criteria, making it easier to analyze. Filtering hides rows or columns that do not meet the specified criteria, allowing for a focused view on the data that matters.

Excel supports Basic Math functions like addition, subtraction, multiplication, and division, which can be applied directly in cells to perform quick calculations and complex arithmetic operations, facilitating extensive data analysis and financial modeling.

Excel Charts transform rows of data into visualizations that make it easier to understand trends, outliers, and patterns in data. With a variety of chart types like pie, line, bar, and scatter, Excel allows users to represent data graphically for impactful presentations and reports.

Excel Formulas are expressions used to calculate and process data. They range from simple mathematical operations to complex functions that analyze financial, statistical, or engineering data. Formulas increase efficiency and accuracy in data handling.

Access Concepts

Access facilitates the seamless importing and exporting of data to and from other Microsoft Office applications, enhancing data integration and interoperability across platforms.

Microsoft Access is well-suited for individual users or small teams due to its user-friendly interface and powerful database management capabilities, making it ideal for managing personal databases or small business data.

Access is a powerful tool for creating and managing databases. It offers a comprehensive set of database tools for data entry, querying, and running reports. Users can easily design database structures and establish relationships between data points.

Access allows users to publish databases on the web. This feature enables databases to be shared and accessed online, which facilitates collaboration and remote data management.

The "Tell Me" feature in Access acts as an intelligent assistant, helping users navigate the features of the application and quickly find the tools needed to complete tasks efficiently.

Report View in Access allows users to create professional, formatted reports based on the data stored in their databases, suitable for presentations or print.

PPT Concepts

PowerPoint offers robust formatting tools that let you modify the appearance of slides, including fonts, colors, and paragraph settings, to enhance the visual appeal of presentations.

Presentation Playback features in PowerPoint allow for the seamless playing of slideshows, with options to include timed transitions, narrations, and animation to engage the audience effectively.

Backstage View in PowerPoint provides quick access to file-related functions such as save, open, print, and share. It is essential for managing file properties and preparing presentations for distribution.

SmartArt Diagrams in PowerPoint allow users to visually communicate information. Users can choose from a variety of diagrams to enhance message clarity and engagement, including hierarchical structures, cyclic processes, and lists.

Transitions and Animations are tools within PowerPoint that add motion to slides and elements within them, helping to keep the audience focused and making the presentation more dynamic and interesting.

The Cropping tool in PowerPoint allows users to trim and remove unwanted outer areas from images directly within the presentation, helping to focus the viewer's attention on the desired part of the image.

Word Concepts

Styles in Word provide a quick way to apply a set of formatting choices consistently throughout your document. With styles, you can easily manage the formatting of your text and ensure consistency across your document.

Templates in Word are pre-designed documents that help you start your projects quickly. They often include pre-set layouts, styles, and formatting, serving as a framework for your documents.

Contextual Tabs in Word appear on the Ribbon only when you need them, related to the specific task or feature you're using, such as formatting tables or pictures.

Nonprinting characters in Word, like paragraph marks and spacing symbols, help users understand document formatting but do not appear in the printed document.

Edit Mode in Word allows you to make changes to your document, providing tools for text insertion, deletion, and formatting.

Tab Stops and the Tab Key in Word are used to create consistent indentation and alignment in your document, facilitating the organization of text and enhancing readability.
